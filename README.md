# 1、快速开始
通过git下载最新项目
git clone https://github.com/maurosoria/dirsearch.git
安装依赖
pip3 install -r requirements.txt
# 2、下载插件并设置插件
设置dirsearch.py位置
<img width="1152" alt="图片" src="https://user-images.githubusercontent.com/35289748/116520482-5e7b4680-a905-11eb-95b5-e41a936e6b7c.png">
设置python位置
注意：需要使用python3，默认值为python3，Windows下请确认cmd窗口输入python，能够调用python3，或者设置成可以调用python3的命令。
<img width="980" alt="图片" src="https://user-images.githubusercontent.com/35289748/116520801-c598fb00-a905-11eb-87a7-1caef9c1bdfa.png">
此时应该设置python。

通过设置以上参数，我们可以正常使用插件啦
